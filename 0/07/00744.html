Bahan:
  • 200 gr bawang bombai, iris melintang tipis
  • 200 gr paprika hijau, iris bentuk korek api
  • 200 gr wortel, iris bentuk korek api tipis
  • 100 gr daun bawang, iris bentuk korek api
  • minyak untuk menggoreng
Adonan tempura:
  • 150 gr tepung tempura
  • 1 btr telur
  • 250 ml air es
  • 1/2 sdt garam
  • 1/2 sdt lada halus
Saus teriyaki:
  • 125 ml kecap asin
  • 100 gr gula palem
  • 50 ml kaldu ayam
Cara membuat:
  1. Campur semua bahan adonan tempura aduk rata.
  2. Masukkan bawang bombai, paprika, wortel, daun bawang, aduk rata.
  3. Panaskan minyak, goreng adonan 1 sdm hingga kuning kecokelatan.
  4. Angkat, sajikan tempura dengan sausnya.
  5. Saus: campur semua bahan saus.
  6. Jerangkan di atas api kecil hingga mendidih. Angkat.
Untuk 8 porsi
